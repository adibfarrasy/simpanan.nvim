===================================================================================
                                                                *simpanan.nvim*

This plugin is a datastore client that returns a JSON payload for convenient
data processing. 

                                                       *simpanan.list_connections()*
simpanan.list_connections()

    Display a popup for managing connections.    >
        
        simpanan.list_connections()
<

                                                               *simpanan.execute()*
simpanan.execute()

    Execute the highlighted text in visual mode and execute the query. >

        simpanan.execute()
<

                                                       *simpanan.refresh_schemas()*
simpanan.refresh_schemas({label})

    Drop the on-disk schema cache for one connection (or all connections
    when {label} is nil or "") so the next autocomplete request
    re-introspects. Useful after running a migration in another tool
    when you don't want to wait up to an hour for the automatic refresh. >

        simpanan.refresh_schemas("pg0")  -- just this connection
        simpanan.refresh_schemas()        -- every connection
<
